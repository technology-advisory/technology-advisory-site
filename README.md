# Technology Advisory

Arquitectura e infraestructura crítica en entornos híbridos on-premises y cloud.

## Estructura

```
.
├── index.html              # Página principal
├── css/
│   ├── global.css          # Variables y base styles
│   └── index.css           # Estilos del home
├── js/
│   ├── menu.js             # Menú dinámico
│   ├── footer.js           # Footer dinámico
│   └── main.js             # Lógica principal
├── sobre-mi/
│   └── sobre-mi.html       # Página de perfil
└── legal/
    └── legal.html          # Página legal
```

## Características

- **Menú limpio** al estilo FraudeDigital con SVGs
- **Hero responsive** con 5 pilares de infraestructura
- **Footer dinámico** con 3 columnas
- **Modal "En desarrollo"** para secciones no implementadas
- **Colores corporativos**: Verde #059669, Naranja #f97316

## Colores

- **Primary**: `#059669` (Verde)
- **Accent**: `#f97316` (Naranja)
- **Dark**: `#0f172a` (Azul marino)
- **Light**: `#f8fafc` (Gris claro)

## Desarrollo

Las secciones de Arquitectura, Operaciones, Seguridad y Gobernanza mostrarán un modal "En desarrollo" cuando se haga clic.
